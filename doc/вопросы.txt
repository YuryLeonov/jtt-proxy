1) Нужно знать количество спутников, с которым система сейчас работает, так как пилот смотрит на это значение и, возможно, отбрасывает события с недостаточным количеством(пока это значение в конфиге).
2) Нужно для каждого события знать направление движения ТС.Это число от 0 до 359, где 0 - север и далее по часовой стрелке.
3) Периодически нужно отправлять сообщение местоположения на платформу. Для этого мне нужно запрашивать из базы текущие скорость, направление, высоту над уровнем моря, координаты и статус терминала(это как сообщение аларма, но без параметров аларма).

Параметры статуса терминала:

"ACC" : "on/off"    -    повернут ли ключ зажигания
"Positioned" : true/false    -    используется ли система позиционирования
"latitude" : "south/north"    -    какая широта
"longitude" : "west/east"    -    какая долгота
"OperationStatus" : "on/off"    -    статус работы(не до конца понял что имеется в виду)
"CoordinatesEncryption" : true/false    -    зашифрованы ли значения координат
"LoadLevel" : 0,1,10,11    -    0(ТС пусто), 1(ТС заполнено наполовину), 10(зарезервировано), 11(полная загрузка)
"VehicleOilCircuitStatus" : "on/off"    -    статус работы масляного контура
"VehicleCircuitStatus" : "on/off"    -    статус работы главной электрической цепи автомобиля
"DoorStatus" : "locked/unlocked"    -    заблокированы ли двери
"IsFrontDoorOpened" : true/false    -    открыта ли передняя дверь(пассажирская)
"IsMiddleDoorOpened" : true/false    -    открыта ли средняя дверь
"IsBackDoorOpened" : true/false    -    открыта ли задняя дверь
"IsDriverDoorOpened" : true/false    -    открыта ли передняя дверь(водителя)
"IsFifthDoorOpened" : true/false    -    открыта ли дополнительная пятая дверь
"GPSStatus" : true    -    используется ли GPS
"BeidouStatus" : true/false    -    используется ли BEIDOU
"GlonassStatus" : true/false    -    используется ли GLONASS
"GalileoStatus" : true/false    -    используется ли Galileo
"satellitesCount" : 3   -   число подключенных спутников
